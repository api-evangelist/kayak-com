# kayak-com

KAYAK — travel meta-search for flights, hotels, cars, packages, and cruises (Booking Holdings).

KAYAK aggregates fares and inventory from hundreds of online travel agencies, airlines, hotel chains, and car-rental providers into a single comparison surface. It is part of Booking Holdings (NASDAQ: BKNG) and operates alongside sister brands HotelsCombined and Momondo, plus KAYAK for Business for corporate travel.

There is no public self-serve developer API. Programmatic access is limited to vetted partners through the KAYAK Affiliate Network and KAYAK Partner Network — typically deep-link builders, co-branded and white-label search pages, XML/JSON feeds, and conversion tracking, all gated behind a partner agreement. The KAYAK GitHub org contains internal tooling and forks rather than client SDKs.
